# Licensing and Attribution

THC Methodology is licensed under the MIT License.

The goal is low-friction adoption. Projects should be able to copy, adapt,
embed, and automate the methodology without needing custom permission.

## What This Allows

Projects may use THC Methodology for:

- grading rubrics
- local audits
- agent skills
- project templates
- internal review processes
- public reports
- tooling and websites
- commercial or non-commercial use

See `LICENSE` for the full legal terms.

## Credit

Credit is appreciated where practical, especially when THC is reused in public
docs, tools, reports, courses, or products.

Recommended credit:

```txt
Uses THC Methodology, created by Steven Pajewski / Vel Labs.
```

Short credit:

```txt
Based on THC Methodology.
```

Credit should not imply endorsement, certification, approval, or review by
Steven Pajewski or Vel Labs unless that review actually happened.

## Public Claims

A project may say it uses THC Methodology.

A project should not claim:

- certification
- guaranteed reliability
- security approval
- Vel Labs review
- independent public score

unless the supporting review artifact exists.
